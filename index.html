S.H.O.P: Simplified Harbour-Witnessing Online Platform

The Goal:

To transition the functioning elements of the Harbour Witnessing excel spreadsheet while removing the pitfalls to provide the best user experience possible.

Requirements

- Needs to be free

- Provide collaboration that can be seen by all users

- Allows new ideas or systems to be implemented without starting from scratch

- Needs to be multi device compatible

- Provides Instructions within the project on how to use it

- Needs to be able to implement automation for various tasks

Problems

- Unsure what the better platform would be to use a program or a website

- Finding free options limits functionality

- I have little to no idea as to the preferences of those who will be using the platform

Potential Solutions

Google AppsScript/Sheets

- Can be a webApp (slightly worse than a website), or any of googles other apps

- Limited Use with how you can write code e.g. less functions

- So long as the user is familar with googles UI should have no difficulty using system

Amazon Web Services

- Option to be a website or a program

- Virtually no code limits (only the amount) shouldn't be an issue with the scale we're working at

- Is the most complicated of the 3 but has the most flexiblitiy

GitHub

- Only a website (one page) you have to pay for more

- No code limitations on how it's written & interpreted

- Can act as a user friendly hub to various websites and applications

My Thoughts

We could use GitHub to create 1 central hub/homepage which can be easy to access and edit on all devices. We could add links to webApps or google sheets which could act like the original excel spreadsheet except it would be multi device compatible and live so multiple people can edit in the same document that way it avoids confusion as to where to find documents. Another possibiltiy would be to use Amazon Web Services in a similar way, I still need to do more reasearch to find what it's potential is and whether it's free.

I've also kept in mind the comment about automatically emailing the ships, I've suggested these platforms because they are capable of an idea like that and it should be relatively easy to setup with time.
